Intellij 15가지 중요 키





    Command 설명 화면 Command 설명 화면 Command 설명 화면

  1. [Alt + Enter]

    		2.

    • 에러를 수정할 수 있다.
    • 경고 및 오류코드 위에 커서를 한 뒤, [Alt + Enter]를 통해 경고 및 오류를 수정할 수 있다.
    • 더 짧은 코드로 바꿔준다.

  2. [F2]

    		3.

    • 오류, 경고, 제안으로 이동
    • [F2]로 이동을 하고, [Alt + Enter]로 코드 수정 가능

  3. [⌘1]
    [Alt + 1]

    		4.

    • 프로젝트 창 열기 및 초점 이동
    • 방향키로 초점 이동
    • 타이핑을 통한 검색 가능

  4. [Esc]

    		5.

    • 프로젝트 창에서 editor로 초점 이동
    • 변경사항을 저장하기 않고 저장

  5. [⌘E]
    [Ctrl + E]

    		6.

    • 가장 최근에 열었던 파일 확인 가능
    • 방향키로 파일 탐색 가능
    • 키보드 단축키가 없는 도구창을 열 수도 있다.
    • 키워드를 통해 특정 항목 검색 가능

  6. [⌘B]
    [Ctrl + B]

    		7.

    • 심볼이 선언된 위치로 이동
    • [⌥+⌘+B] / [Ctrl+Alt+B]로 구현탐색 가능

  7. [Alt + F7]

    		8.

    • 해당 심볼이 사용되는 모든 위치 표시

  8. [Ctrl twice]

    		9.

    • Run Anything창으로 어떤 것이든 실행가능
    • default로 가장 최근에 실행한 실행 구성이 목록에 표시됨
    • 실행할 항목의 이름 입력 가능

  9. [⌥↑ / ⌥↓]
    [Ctrl + w]/
    [Shift + Ctrl + w]

    		10.

    • [⌥↑] / [Ctrl + w]커서 주변의 코드 선택 단어 단위로 섹션 확장 가능
    • [⌥↓] / [Shift + Ctrl + w] 축소

  10. [⌘/]
    [Ctrl + /]

    		11.

    • 커서 문장 주서처리
    • 주석 문장에 사용시 주석 해제
    • [⌥+⌘+/] / [Shift+Ctrl+/] 선택 블록 전체 주석

  11. [⇧⌘↲]
    [Shift + Ctrl + Enter]

    		12.

    • 사용시 세미콜론 추가 가능
    • 중괄호 추가 및 적정 위치에 커서 위치

  12. [⌥⌘L]
    [Ctrl + Alt + L]

    		13.

    • 현재 파일의 서식을 프로젝트 표준에 맞춰 정렬
    • [⌥⌘+Shift+L] / [Shift+Ctrl+Alt+L] 옵션선택 가능

  13. [^T]
    [Shift + Ctrl + Alt + T]

    		14.

    • 모든 리팩토링 옵션 표시
    • 방향키와 엔터로 옵션 선택
    • 단축키가 존재할 경우 단축키 역시 표시

  14. [↑⌘A]
    [Shift + Ctrl + A]

    		15.

    • 모든 액션 검색
    • 단축키 역시 표시
    • 액션 및 설정 변경 가능
    • 도구창 검색 및 열기 지원

  15. [Shift twice]

    		16.

    • 모든 것을 검색
    • 설정 변경 가능
    • 기본적으로 최근 파일을 표시하므로 [Ctrl+E]대체 가능
    • 클래스, 파일, 심볼 및 액션 검색 결과 모두 표시
    • 명령어 지원
